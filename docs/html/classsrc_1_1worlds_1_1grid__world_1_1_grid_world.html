def src.worlds.grid_world.GridWorld.load_grid_from_file (   self,
  filename 
)

Creates an entire gridworld model by loading a file.

File must contain a grid of x,y rows and columns. Each entry must be 'B', 'G', 'R', 'W', 'Y' where 'B' => black (invalid) state, 'G' => green (goal) state, 'R' => red (trap) state, 'W' => white (normal) state, 'Y' => yellow (start) state.

Parameters
filenamethe name of the file (e.g. grid.txt)
rewardoptional keyword that sets reward for white (normal) states globally
trap_valueoptional keyword that sets reward for red (trap) states globally
goal_valueoptional keyword that sets reward for green (goal) states globally
probabilitiesoptional keyword that holds a dictionary with the following values: self.Action.UP => probability of going forward, self.Action.RIGHT => probability of going right, self.Action.LEFT => probability of going left, self.Action.DOWN => probability of going backward
