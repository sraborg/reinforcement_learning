Creates an entire gridworld model by loading a file.

File must contain a grid of x,y rows and columns. Each entry must be 'B', 'G', 'R', 'W', 'Y' where 'B' => black (invalid) state, 'G' => green (goal) state, 'R' => red (trap) state, 'W' => white (normal) state, 'Y' => yellow (start) state.

Parameters
filenamethe name of the file (e.g. grid.txt)
rewardoptional keyword that sets reward for white (normal) states globally
trap_valueoptional keyword that sets reward for red (trap) states globally
goal_valueoptional keyword that sets reward for green (goal) states globally
probabilitiesoptional keyword that holds a dictionary with the following values: self.Action.UP => probability of going forward, self.Action.RIGHT => probability of going right, self.Action.LEFT => probability of going left, self.Action.DOWN => probability of going backward Inner Function that determines whether a (x,y) state is a valid location.

Validates the (x,y) coordinate against the loaded file. If it valid, it also determines its states "type"

Parameters
xstate's x coordinate
ystate's y coordinate
max_index_xthe number of columns
max_index_ythe number of rows
Returns
boolean, type where boolean is true/false depending on if state (x,y) is valid and type is in {'B', 'G', 'R', 'W', 'Y'}
